Adding a Public Peer for RiV Mesh

The RiV Mesh node is a built-in module in CupLink, acting as a mesh node and allowing communication between CupLink users through the Mesh secure network.


Port Availability

If you see this message, it means your port is available and ready to be published for a public peer. If you do not wish to publish it, click the Cancel button. You can enable this port to be published in the Public Peer settings.
